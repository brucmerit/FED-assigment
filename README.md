TRAVEL EXPENSE TRACKER
A simple, lightweight web app for logging and reviewing travel expenses on the go.

Design Process
This web app is designed for travellers to keep track of their expenses during their travel. We want travelling to be an affordable and enjoyable experience.

Provide us insights about your design process, focusing on who this website is for, what it is that they want to achieve and how your project is the best way to help them achieve these things.

Use Cases

As a traveller, I want to be able to log down every expenses that I incur during the trip.
As a user type, I want to perform an action, so that I can achieve a goal.

Edit existing expense - Travellers can edit existing record in the case of error

Delete expense - Travellers can delete exist record 

View total expenses - Travellers can see the total sum of the recorded expenses.
